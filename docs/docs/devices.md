# Devices

Stride has been tested with the following devices:

## Garmin Running Watches

- **Forerunner 10**
- **Forerunner 110**
- **Forerunner 235**
- **Forerunner 745**
- **Forerunner 965**

## Garmin Cycling Computers

- **Edge 530**
- **Edge 840**

## Additional Support

While Stride has been tested with the devices listed above, it can parse and display data from any device that produces standard FIT files. If your device isn't listed but exports FIT files, it should work with Stride.

If you encounter issues with a specific device, please report it on the [GitHub repository](https://github.com/vlegout/stride).
